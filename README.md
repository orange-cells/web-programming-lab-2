_Описание_:

Репозиторий содержит код приложения для создания ToDo-листов.

_Функционал_:

1. Добавление и удаление задачи из списка.
2. Редактирование задач после двойного нажатия на текст.
3. Возможность отмечать задачу как выполненную.
4. Сортировка задач по дате, а также фильтрация по статусу (выполнена/не выполнена/все задачи).
5. Возможность менять порядок задач при помощи drag-and-drop.
6. Поиск задач по тексту.
7. Сохранение задач в localStorage.

_Технологический стек_:

Работа выполнена с использованием HTML, CSS, JavaScript.

_Адаптивный дизайн_:

Обеспечено корректное отображение сайта на разных устройствах (ноутбук, планшет и смартфон).
